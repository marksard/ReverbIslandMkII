# ReverbIslandMkII

8HP Eurorack modular multi Effects processor on Spin Semi FV-1

![module_01](/_data/module_01.jpg)  

## Specification

|Name|Description|
|---|---|
| Power Supply | Eurorack ±12V (2x5pin)|
| Width | Eurorack 8HP |
| Depth | 300mm (Depth from panel) |
| Microcontroller | RP2040 (Waveshare RP2040 ZERO) |
| IN L, IN R | Equipped with L-input attenuator<br>R-input is preset-dependent (connected to L-input when disconnected)<br>Impedance:100kΩ |
| P0~P3 CV IN | ±5V with attenuator<br>Impedance:100kΩ |
| OUT1, OUT2 | Impedance:1kΩ |
| Controllers | P0~P3: FV-1 Preset Control<br>Mode,-,+:Preset Selector |
